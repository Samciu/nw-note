# nw-note
note-webkit实践

# 使用说明：

桌面版下载后，运行`nw-note.exe`即可  

[在线预览Demo](http://samciu.github.io/note/)  

# nw.js介绍
基于 `Chromium` 和 `Node.js`，`webkit` 提供 DOM 操作，`Node.js` 提供本地化操作。  

`NW.js`将二者的context完全整合，能在 `DOM` 上调用所有 `Node.js` 模块。
